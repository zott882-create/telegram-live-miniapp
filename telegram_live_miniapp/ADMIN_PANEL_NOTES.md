# Telegram-админ-панель

ID владельца встроен в сборку: `851766591`.

После развёртывания откройте чат с ботом и отправьте `/admin`.

Получение команд:
- на Render, Railway, Koyeb и другом сервере с публичным HTTPS — защищённый Telegram webhook;
- локально или без публичного HTTPS — резервный `getUpdates` polling.

Доступные разделы:
- статистика пользователей, LIVE-матчей, найденных матчей и времени сбора;
- пользователи, блокировки и доступ;
- выдача подписок на 1/7/30/90/365 дней, произвольный срок или навсегда;
- проверка и отзыв подписки;
- включение обязательного доступа по подписке;
- очередь уведомлений, состояние системы и лимит пользователей онлайн.

Команды администратора: `/admin`, `/stats`, `/users`, `/subs`, `/system`, `/id`.
Команды пользователя: `/subscription`, `/id`.

Обязательная переменная: `BOT_TOKEN`.
Если платформа не передаёт публичный адрес автоматически, задайте:
`PUBLIC_BASE_URL=https://ваш-домен`
