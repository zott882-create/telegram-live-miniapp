MATCH_CENTER_README

Изменения:
- Удалён главный экран Match Radar / лента умных статусов из пользовательского интерфейса.
- Старой текстовой аналитической вкладки нет.
- Главная снова показывает обычный список матчей с простыми вкладками: Все, 1Т, 2Т, Голы.
- Внутри каждого матча добавлен визуальный Match Center:
  1) 📊 Центр — общий статус матча, темп, инициатива, ключевые причины, быстрые метрики.
  2) ⚽ События — timeline голов и карточек + график давления за последние 20 минут.
  3) 📈 Статы — live-статистика, пульс матча и давление.
  4) 🔥 Форма — форма команд и последние матчи.
- Value calculator не используется.
- Внешний GPT/API для AI не используется.
