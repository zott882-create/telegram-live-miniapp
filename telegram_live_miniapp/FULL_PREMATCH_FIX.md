# V10.3 — полный список прематчей

Исправлена причина, по которой Mini App показывал только 31–35 матчей при большем счётчике на SportScore.

Что изменено:
- основной источник прематчей теперь — датированные страницы SportScore;
- загружаются сегодня, завтра и следующий день (`SPORTSCORE_PREMATCH_DAYS_AHEAD=3`);
- убрана зависимость от SEO JSON-LD, где находится только небольшая выборка матчей;
- парсер читает все отрисованные карточки независимо от текущего CSS-класса строки;
- поддержаны `href`, `data-url`, `data-match-url`, `data-fixture-slug` и `data-match-slug`;
- добавлена диагностика `source_stats` в ответ `/api/prematch`;
- публичный API SportScore с лимитом 50 используется только как дополнительный источник, а не как полный календарь.

После загрузки на Render выполнить: Manual Deploy → Clear build cache & deploy.
