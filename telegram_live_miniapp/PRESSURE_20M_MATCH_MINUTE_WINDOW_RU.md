# v9.92 — давление за 20 минут по минутам матча

Изменение:

- Раньше карточка «Давление за 20 мин» считалась по времени collector.
- Теперь она считается по игровой минуте матча.
- На 21-й минуте из окна выпадает 1-я минута.
- На 25-й минуте из окна выпадают 1–5 минуты.
- В 60' окно примерно 40–60 минут.
- Если старые сохранённые снимки ещё не содержат минуту матча, код безопасно использует старый режим по времени collector.

Railway variables менять не нужно.
