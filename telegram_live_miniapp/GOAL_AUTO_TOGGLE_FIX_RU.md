# v9.91 — переключатель авто-уведомлений о голе

Исправлено поведение уведомлений:

- Раньше матч, найденный через «Ждём гол», автоматически подписывался на уведомление о следующем голе.
- Теперь это можно переключать отдельно.
- По умолчанию авто-голы выключены.
- Вкладка «Ждём гол» получила второй переключатель: **Гол после найденного матча**.
- Если переключатель OFF, бот отправляет только уведомление «Ждём гол», но не шлёт потом автоматический гол по этому матчу.
- Ручные гол-уведомления по звёздочке/избранному не ломаются.

Railway variables менять не нужно.
