# Карточки для отправки матчей в Telegram

Добавлено:

- кнопка **ОТПРАВИТЬ В TELEGRAM** на экране матча;
- страница `/share/match?id=<match_id>` с Open Graph meta-тегами;
- PNG-карточка `/share/card.png?id=<match_id>` в стиле LIVE TICKET;
- зависимость `Pillow` для генерации изображения.

## Важно для деплоя

На Render/сервере укажите переменную окружения:

```
PUBLIC_BASE_URL=https://ваш-домен.onrender.com
```

Без публичного HTTPS-домена Telegram не сможет подтянуть превью-картинку.

После деплоя откройте любой матч и нажмите **ОТПРАВИТЬ В TELEGRAM**. В чат уйдёт ссылка, а Telegram покажет красивую карточку-превью матча.
