# v9.93 — быстрыее скрытие завершённых матчей

Исправлено:

- Матчи, которые закончились или пропали из live-ленты провайдера, больше не должны висеть 10–15 минут.
- По умолчанию live-cache скрывает матч примерно через 3 минуты после последнего появления у провайдера.
- Матчи с минутой 99+ скрываются из live-ленты быстрее.
- История для уведомлений/деталей сохраняется отдельно, чтобы старые Telegram-кнопки не ломались.

Новые настройки, если нужно вручную:
- `STALE_LIVE_HIDE_AFTER_MINUTES=3`
- `LIVE_FEED_MAX_MINUTE=99`
- `FINISHED_LIVE_FEED_GRACE_SECONDS=180`

Railway variables можно не менять: defaults уже обновлены.
