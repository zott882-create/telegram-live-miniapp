# v9.84 — фикс завершённых матчей и графика давления

Что исправлено:

1. Матчи на 100+ минуте больше не висят в live-ленте, если провайдер ещё несколько минут отдаёт их как live.
   - Добавлена настройка `LIVE_FEED_MAX_MINUTE`.
   - Значение по умолчанию: `100`.
   - То есть матчи с минутой `100'`, `101'` и выше скрываются из главного списка.

2. График давления нормализован по таймам:
   - 1-й тайм занимает левую половину графика.
   - 2-й тайм занимает правую половину графика.
   - К 90+ минуте график занимает всю ширину.
   - Бары больше не выглядят с большими пробелами из-за редких снимков collector.

Railway заново не настраивать. Просто обнови файлы в том же GitHub repo и сделай redeploy.

Рекомендуемые переменные Railway:

```env
DATA_MODE=auto
LIVE_FEED_MAX_MINUTE=100
STALE_LIVE_HIDE_AFTER_MINUTES=10
COLLECTOR_INTERVAL=90
COLLECTOR_MAX_MATCHES=350
COLLECTOR_WORKERS=1
COLLECTOR_REQUEST_DELAY=1.0
ADMIN_POLLING_ENABLED=0
ONLINE_USER_LIMIT=500
```
